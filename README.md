🎡 LunaPark Management System
A modern 3-Tier Web API for managing amusement park operations, from attractions to ticketing.

🚀 Key Features
Attraction Management: Full CRUD operations for park facilities.

Smart Ticketing: Scalable logic for purchasing and retrieving tickets.

Employee Tracking: Organized management of park staff.

🛠 Tech Stack
Backend: ASP.NET Core Web API.

Database: SQL Server via Entity Framework Core (DB First).

Mapping: AutoMapper for clean DTO transitions.

Architecture: Decoupled 3-Layer structure (BLL, DAL, DTO).

🏗 Architecture Overview
UI (Web API): Handles HTTP requests and Controllers.

BLL (Business Logic): Validations and core park logic.

DAL (Data Access): Direct communication with the Database.
